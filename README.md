# Build a Formik Form

Week 13: Building a Formik Form.

This project revolves around building a form with the Formik library. By collecting information, Formik keeps things neat and tidy, by making testing, refactoring, and reasoning your forms with ease.

With this project you can submit Email and Password credentials, or be prompted to fill in the field as needed. 


## Installation

To install this project on your PC or Mac, you need to run the following instructions in terminal.

Navigate to the root file of the project, or starting files given, and run the following:
```bash
  npm install 

  npm start
```
After the commands are executed, the App will be locally hosted and open in a default browser.
